--- README ---

Esta é uma classe que exporta dados de uma matriz para um arquivo CSV. 
Nela, você pode expecificar alguns atributos, tais como o delimitador do CSV e/ou
diretório temporário cujo qual será salvo o CSV para que o usuário faça o download do mesmo.

Caso os atributos não sejam especificados, a classe usará valores padrões pré-definidos
no construtor da classe.

Em alguns ambientes Linux, é necessário especificar um diretório cujo qual a classe
terá permissão de escrita no mesmo, por exemplo "/home/user/"

Por favor, verifique o arquivo "exemplo.php" para mais informações técnicas.
